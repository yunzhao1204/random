# 无限暖暖 - 随机衣橱生成器

## 项目简介

这是一个专为《无限暖暖》游戏玩家设计的随机衣橱生成器工具。它基于游戏内实际数据，帮助玩家随机选择服装搭配，增加游戏体验的趣味性和多样性。

## 功能特性

### 核心功能
- 提供游戏中所有服装类别的完整列表
- 支持按类别选择需要包含在随机生成中的服装类型
- 根据所选类别生成对应的随机服装序号
- 实现了游戏内的服装搭配规则验证

### 用户界面特点
- 响应式设计，支持桌面端和移动端访问
- 美观的渐变色彩主题，采用柔和的紫色和粉色为主色调
- 手风琴式折叠菜单，方便浏览和管理大量子分类
- 实时统计信息展示面板
- 流畅的动画效果和交互反馈

### 服装分类覆盖
- 发型
- 连衣裙、上衣、下装等主要服装部件
- 外套、袜子、鞋子等配件
- 饰品（发饰、帽子、耳饰、颈饰等11个子分类）
- 妆容（底妆、眉妆、睫毛、美瞳等6个子分类）
- 闪光祝福（头部、手部、脚部）

## 使用方法

1. 在表格中勾选你想要包含在随机生成中的服装类别
2. 可以通过"全选"或"全不选"按钮快速操作
3. 点击"生成随机序号"按钮，系统会根据所选类别生成对应的随机服装序号
4. 生成的结果按照数值大小排序，并标注其在推荐搭配中的行列位置

## 技术实现

### 前端技术栈
- HTML5
- CSS3（使用 TailwindCSS 框架）
- JavaScript（原生实现，无框架依赖）
- FontAwesome 图标库

### 核心逻辑

#### 数据结构
所有服装类别存储在一个数组对象中，每个类别包含：
- 主分类和子分类名称
- 该类别服装总数
- 是否被选中状态
- 特殊分组标识（互斥组、依赖组等）

#### 规则约束
1. **互斥规则**：连衣裙与上衣/下装互斥，不能同时选择
2. **依赖规则**：上衣和下装必须同时选中或同时不选中
3. **有效性验证**：必须穿着连衣裙或完整的上衣下装才能生成有效搭配

#### 交互机制
- 手风琴效果：饰品、妆容、闪光祝福三个大类采用折叠显示方式
- 半选状态：父级分类支持半选状态表示部分子类被选中
- 实时统计：左侧统计面板实时显示总类别数和已选类别数
- 输入验证：对用户输入的最大值进行边界检查

## 开发细节

### 代码组织
- 所有HTML、CSS和JavaScript代码集中在一个HTML文件中
- 使用模块化函数组织JavaScript逻辑
- CSS采用TailwindCSS实用类结合自定义样式的方式

### 响应式适配
- 针对移动设备进行了专门优化
- 表格在小屏幕上自动横向滚动
- 控件布局在不同屏幕尺寸下自适应调整
- 统计面板在移动端变为底部抽屉式交互

## 使用说明

1. 打开 `随机衣橱.html` 文件即可使用
2. 根据需要勾选相应的服装类别
3. 点击"生成随机序号"按钮获得搭配建议
4. 结果会按照数值从小到大排列，并标注所在行列位置

## 注意事项

- 本工具基于2025年11月20日的游戏数据
- 工具仅供个人娱乐使用，与游戏官方无关
- 遵循游戏内服装搭配规则，确保生成结果的有效性
